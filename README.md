## Project Description
This program is a console-based inventory and purchase order management system created in Python. The system allows a business to manage products, vendors, inventory levels, purchase orders, shipments, and reports. Data is saved and loaded using JSON files.

## Features
- Add, view, search, edit, and deactivate products
- Add, view, search, and edit vendors
- Create and manage purchase orders
- Receive shipments and update inventory
- Generate inventory and purchase order reports
- Save and load data using JSON
- Automatic reorder suggestion system

## Files
- main.py
- models.py
- inventory_manager.py
- file_manager.py
- reports.py
- README.md
- reflection.md
- code_explanation.txt

## Extra Feature
The program includes an automatic reorder suggestion system that identifies low-stock items and recommends reorder quantities.
